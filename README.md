# Treinamento de Protecao

Aplicacao React criada com Vite para treinamento operacional de protecao de
subestacoes.

## Requisitos

- Node.js 20.19 ou superior
- npm

## Desenvolvimento

Instale as dependencias:

```bash
npm install
```

Inicie o servidor:

```bash
npm run dev
```

Abra o endereco exibido pelo Vite no terminal, normalmente
`http://localhost:5173`.

## Build

```bash
npm run build
```

Para visualizar o build de producao:

```bash
npm run preview
```

## Compatibilidade com Bolt

As versoes do Vite e do plugin React estao fixadas no `package.json` para
evitar incompatibilidade do Vite 8/Rolldown com o WebContainer do Bolt.
